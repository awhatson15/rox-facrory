# Unpacking

> Исследование даёт сырьё. Распаковка превращает его в готовые продающие тезисы.

## Что это

**Распаковка** — процесс перевода данных из исследования в:
- Обещания для продукта
- Тезисы о сервисе
- Отработку возражений
- Киллер-фичи
- Буллеты

Результат: карта смыслов из 8 столбиков, готовая для лендингов/прогревов/контента.

## Формула обещания

```
ТЫ + глагол (обещание результата) + как это будет реализовано
```

### Правило
Каждый тезис должен быть **ЖИРНО и СОЧНО** обоснован.
Чтобы хотелось купить даже от одной фразы.

### Пример

❌ **Плохо:**
```
"Ты внедришь услугу и станешь зарабатывать больше"
```

✅ **Хорошо:**
```
"Ты внедришь дополнительную услугу микроворонка
и минимум в 3 раза увеличишь свой средний чек.

Благодаря этому инструменту ты сможешь приводить 
клиенту прогретых клиентов, которые сразу готовы покупать."
```
