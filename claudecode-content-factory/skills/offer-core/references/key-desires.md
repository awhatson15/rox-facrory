# Как выделить ключевые желания

## Проблема

Из исследования получаешь 10-15 желаний.
Для ядра нужно 3 КЛЮЧЕВЫХ.

Как выбрать?

---

## Критерии ключевого желания

### ✅ Ключевое
- Ради этого люди РЕАЛЬНО приходят
- Про личную выгоду (деньги, время, результат)
- Вызывает сильную эмоцию
- Понятно всем в нише

### ❌ НЕ ключевое
- Побочная фича
- Про процесс, а не результат
- Второстепенная польза
- Понятно только части аудитории

---

## Примеры сортировки

### Ниша: Таргет

| Желание | Ключевое? | Почему |
|---------|-----------|--------|
| Зарабатывать 150к/мес | ✅ ДА | Главная выгода |
| Чек от 30к | ✅ ДА | Про деньги |
| 2 часа в день работы | ✅ ДА | Про время |
| Настраивать без банов | ❌ НЕТ | Про процесс |
| Упакованный профиль | ❌ НЕТ | Второстепенно |
| Доступ к чату | ❌ НЕТ | Сервис |
| Личные разборы | ❌ НЕТ | Сервис |

**Вывод:** В ядро идут первые 3.

### Ниша: Похудение

| Желание | Ключевое? | Почему |
|---------|-----------|--------|
| Минус 10 кг | ✅ ДА | Главный результат |
| Сексуальное тело | ✅ ДА | Эмоциональная выгода |
| Энергия и уверенность | ✅ ДА | Состояние |
| Не считать калории | ❌ НЕТ | Про процесс |
| Готовые рецепты | ❌ НЕТ | Сервис |
| Чат единомышленников | ❌ НЕТ | Сервис |

## Тест "А зачем пришёл?"

Спроси себя: "Зачем человек РЕАЛЬНО пришёл?"

```
Человек пришёл на курс по таргету.

Зачем?
— Чтобы научиться настраивать рекламу? НЕТ.
— Чтобы получить сертификат? НЕТ.
— Чтобы зарабатывать деньги, тратя мало времени? ДА.

→ Ключевые желания = про деньги и время.
```

```
Человек пришёл на курс похудения.

Зачем?
— Чтобы научиться считать калории? НЕТ.
— Чтобы получить рецепты? НЕТ.
— Чтобы выглядеть сексуально и чувствовать себя уверенно? ДА.

→ Ключевые желания = про внешность и состояние.
```

## Матрица приоритизации

| | Высокая эмоция | Низкая эмоция |
|---|----------------|---------------|
| **Про результат** | ⭐ КЛЮЧЕВОЕ | Возможно ключевое |
| **Про процесс** | Второстепенное | В ядро НЕ идёт |

## Процесс выбора

### Шаг 1: Выписать все желания из исследования
```
1. Зарабатывать 150к
2. Чек от 30к
3. 2 часа работы
4. Без банов
5. Упакованный профиль
6. Стабильный доход
7. Не торгующиеся клиенты
...
```

### Шаг 2: Отметить что про результат/выгоду
```
1. Зарабатывать 150к ✓
2. Чек от 30к ✓
3. 2 часа работы ✓
4. Без банов
5. Упакованный профиль
6. Стабильный доход ✓
7. Не торгующиеся клиенты ✓
```

### Шаг 3: Выбрать 3 самых эмоциональных
```
1. Зарабатывать 150к → ⭐
2. Чек от 30к → ⭐
3. 2 часа работы → ⭐
```

### Шаг 4: Проверить тестом "конференция"

## Куда идут НЕ-ключевые

Не выбрасывать! Использовать:

- **Сервисные** → В описание продукта
- **Про процесс** → В отработку возражений
- **Второстепенные** → В буллеты и бонусы

Всё найдёт своё место в unpacking.
