# Формулы из методологии Сердитова

## Ядро оффера

### Главная формула
```
Результативный глагол + Идеальный результат + Скорость + [Технология]
```

### Тест "Конференция"
```
Произнеси фразу на конференции.
Все повернулись с "ЧТО?!" → работает.
Никто не повернулся → переделать.
```

### Пример
```
Зарабатывать 150к ежемесячно, 
освоив технологию привлечения для блогеров и инфобиза.
Трать на работу 2 часа в день, 
получая от 30к, работая с 5-6 клиентами.
```

## Формула обещания

```
ТЫ + глагол (обещание) + как это будет реализовано
```

### Правило
Каждый тезис **ЖИРНО и СОЧНО** обоснован.
Хочется купить от одной фразы.

### ❌ Плохо
```
"Ты внедришь услугу и станешь зарабатывать больше"
```

### ✅ Хорошо
```
"Ты внедришь услугу микроворонка 
и минимум в 3 раза увеличишь средний чек.

Благодаря этому инструменту сможешь приводить 
клиенту прогретых клиентов, сразу готовых покупать."
```

---

## Формула названия урока/модуля

```
Как [действие] + [измеримый результат] + [дополнительная выгода]?
```

### Примеры
```
"Как приносить клиентам результат в 90% случаев 
и сделать его вдвое лучше, 
тратя в 3 раза меньше времени?"

"Что такое микроворонка 
и как она увеличит твой чек минимум в 3 раза?"
```

### Правило
Название должно звучать как отдельный продукт,
а не просто "Урок 1. Введение".

## Формула отработки возражения

```
[Почему так случилось у других]
         ↓
[Почему у нас будет по-другому]
         ↓
[Конкретный механизм решения]
```

---

## Формула оффера tripwire

### Вариант 1: Интрига
```
Узнай как [оффер основного продукта]
```

### Вариант 2: Элементы
```
N шагов/элементов, которые позволят тебе [результат]
```

---

## Формула сервисного тезиса

```
[Что получит] + [в каком формате] + [с какой гарантией качества]
```

---

## Формула безопасности

```
[Что может пойти не так] → [Почему у нас не пойдёт] → [Гарантия]
```

---

## Ключевые vs НЕключевые желания

### ✅ Ключевые (в ядро)
- Про деньги
- Про время
- Про главный результат

### ❌ НЕключевые (в описание)
- Про процесс ("без банов")
- Сервисные ("кураторы")
- Второстепенные ("сертификат")

> Люди думают ТОЛЬКО О СЕБЕ.
> Ключевые = про личную выгоду.

## Формула буллета

```
[Интрига] + [Конкретный инсайт] + [Обещание]
```

## Чек-лист формулировки

- [ ] Есть измеримый результат (цифра/состояние)
- [ ] Есть срок достижения
- [ ] Есть механизм (как)
- [ ] На языке клиента (не твои термины)
- [ ] Нет эфемерных слов ("лучше", "больше", "эффективнее")
- [ ] Проходит тест "конференция"
- [ ] Хочется купить от одной фразы
