# Продающие механики по Золотову

*Источник: Золотов "Лидеры телеги 2.0", модули 5-6*

---

## Конверсионная рекламная подача (5 элементов)

### Структура:

1. **Подтема + проблема из КАЗДЕВов** (заголовок)
   - Одна конкретная подтема, не общая
   - "Почему после еды тяжесть и вздутие?"

2. **Фразы ЦА дословно** (узнавание)
   - "Просыпаешься с плоским животиком, к вечеру — как на 4-м месяце беременности"
   - Такое сложно придумать → люди узнают СЕБЯ

3. **Усиление проблемы** (без решения)
   - "Эти симптомы — 100% следствие вашего питания и образа жизни"
   - "А где причина — там и решение"

4. **Экспертные абзацы** (описание, НЕ решение)
   - Показать знание темы
   - НЕ давать ответ — только обозначить

5. **Ссылка на эксперта + конкретные посты**
   - "Познакомьтесь с каналом [имя]"
   - Описание эксперта
   - Ссылки на конкретные посты (не просто на канал)

**Результат:** подписчик по 60-70 руб, работает раз за разом.

---

## Борьба с выгоранием рекламной подачи

### Цикл тематических недель:

1. Выбрать подтему → написать 5-7 постов на неё
2. Создать вовлечение: "Разберу ваши анализы бесплатно" → люди массово пишут в комменты
3. Закупить рекламу НА ЭТИ ПОСТЫ (старые, с социальным доказательством)
4. Следующий цикл: новая подтема → новые посты → новая подача

### Реклама на старые посты

- Не всегда нужны свежие посты для рекламы
- Пост с 200 комментариями > свежий пост без реакций
- Закупать на посты с высоким вовлечением

## 7 этапов продажи в переписке

### 1. Приветствие
- Персонализированное, доброжелательное
- Если пишут вам: не отвечать как бот
- Если пишете вы: персонализация (заметил канал, увидел пост)

### 2. Перехват инициативы (КРИТИЧНО)
> "Какая цена?" → НЕ отвечать ценой!
> "Пока не понимаю ваш запрос. Давайте сначала разберусь, потом подскажу."

- 99% соглашаются
- Без перехвата: "15К" → "Спасибо, до свидания"
- Люди НЕ умеют задавать вопросы → вы должны быть диагностом

### 3. Вбить крюк
- Прочитал, не ответил 5 часов → "Михаил?"
- Не стесняться: если не пишем → 100% потеря
- Правило: добиться осознанного "да" или "нет"

### 4. Выявление потребности
- Чем больше рассказал → тем сложнее уйти
- Длинные голосовые = клиент привязан к диалогу
- Не будет заново всё это записывать другому

### 5. Фильтрация
- Не хочешь работать → отправь на альтернативу
- **Всегда извлекай деньги:** партнёрка, мини-продукт, видеоурок
- "Тебе пока рано на консультацию, купи видеоурок за 3К"

### 6. Персонализированный оффер
- Шаблоны под разные запросы → адаптировать под конкретного человека
- Пример: "Консультация: 12 бесплатных + 7 платных методов, разбор твоих каналов, памятка по делегированию, 1.5 часа, 20К"
- VS просто "Консультация 20К" → первое в вакууме, второе сравнимо

### 7. Допродажа
- Пришёл за курсом 15К → предложить + консультацию 20К
- Отказался → "Тогда просто покупай курс"
- Всегда пытаться увеличить чек

## Принципы

### Продажа = переход на следующий шаг
- Не "получил деньги"
- А "клиент согласился на следующее действие"

### Добивающие сообщения в боте
- Статья в боте (не в канале) → отслеживание дочитываемости
- Не дочитал → серия сообщений:
  - "В разделе 3 раскрыл [ценность]"
  - "Отзывы на статью: [скрины]"
  - "Через X часов удалится"

### 8 диагностик → 9 продаж
- Конверсия >100% возможна
- Предыдущая аудитория покупает из контента без диагностик
- Сильный контент + сильные продажи = синергия

## Дополнительные скрипты и техники (из курса)

### Регламент диагностики (дословно)
> «Давай не буду тратить твоё время. Сейчас: я задаю вопросы, ты рассказываешь, я пытаюсь понять как помочь. В конце — твои вопросы. Хорошо?»

### Скрипт на «скажи цену»
- Мягко: «Мне надо прикинуть объём, пару минут»
- Если настаивает: «Я продаю время. Не понимаю объём. Назову с запасом: 50К. Может меньше, если пойму задачу»
- **Называть верхнюю часть вилки**: «До 30К» → потом 25К не кажется дорогой

### Ап-селл + даунселл
- Клиент хочет курс 15К → предложить курс + консультация = 35К
- Отказался → «Тогда просто курс» → согласие выше
- Кейс: клиент хотел МК за 20К → предложил +бонус → пропал → цена выросла → купил за 30К → бонус всё равно выслал = +6К

### Ложные vs истинные возражения
- Ложное: «я подумаю» → Скрипт: «Может не подошла цена? Что хотелось бы услышать?»
- Истинное: конкретный факт → работать с фактом
- **После каждого отработанного возражения → попытка закрытия**: «Работаем?»

### Триггер избранности
- «Я новичкам не предлагаю, рано» → работает на 70%
- Источник: Ричард Талер, «Новая поведенческая экономика»

### 7 этапов воронки канала
0. Подписка (% уведомлений, норма от 30%)
1. Вовлечение (реакции 10-40% от охватов)
2. Коммуникация (личные диалоги)
3. Эмоция и ценность (самый важный!)
4. Оффер (таблица возражений)
5. Продажа
6. Бэкэнд (повторные — основные деньги)
