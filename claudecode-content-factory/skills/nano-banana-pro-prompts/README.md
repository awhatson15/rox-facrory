# Nano Banana Pro Prompts

Библиотека из 6000+ промптов для генерации изображений через Midjourney и другие AI-модели.

## Что внутри

Промпты организованы по категориям в папке `references/`:

| Файл | Категория | Кол-во |
|------|-----------|--------|
| `profile-avatar.json` | Аватарки и портреты | 828 |
| `social-media-post.json` | Посты для соцсетей | 4793 |
| `infographic-edu-visual.json` | Инфографика и обучение | 390 |
| `youtube-thumbnail.json` | Обложки YouTube | 129 |
| `comic-storyboard.json` | Комиксы и сториборды | 236 |
| `product-marketing.json` | Маркетинг продуктов | 2549 |
| `ecommerce-main-image.json` | E-commerce фото | 267 |
| `game-asset.json` | Игровые ассеты | 255 |
| `poster-flyer.json` | Постеры и флаеры | 391 |
| `app-web-design.json` | Дизайн приложений | 140 |
| `others.json` | Прочее | 719 |

## Как использовать

Скилл `nano-banana-pro-prompts` автоматически ищет подходящие промпты по вашему запросу. Просто скажите Claude что хотите сгенерировать — он подберёт лучший промпт из библиотеки.

## Связанные скиллы

- **nano-banana/** — методология генерации, Identity Kit
- **laozhang-onboarding/** — настройка API для генерации
