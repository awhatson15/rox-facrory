# OpenClaw Content Factory

> Фабрика Контента через Telegram | Макс Галсон | [galson.pro](https://galson.pro)

AI-бот в Telegram, который пишет контент в твоём стиле — по проверенным методикам.

---

## Быстрый старт

1. **Установи OpenClaw** → [Гайд](https://galson.pro/guides/openclaw-telegram)
2. **Скопируй скиллы:**
   ```bash
   cp -r skills/* ~/.openclaw/workspace/skills/
   cp -r workspace/* ~/.openclaw/workspace/
   openclaw gateway restart
   ```
3. **Напиши боту:** `начать`
4. **Ответь на 3 вопроса** → получи первые посты

## 15+ скиллов

| Скилл | Что делает |
|-------|-----------|
| threads | Посты для Meta Threads + Notion шаблон + автопостинг |
| youtube | Сценарии, SEO, хуки, удержание |

| storytelling | Сторителлинг с психологическими триггерами |
| offer | Формулировка оффера (Hormozi) |
| customer-research | Распаковка смыслов, кастдев |
| launch | Система запусков (PLF) |
| nano-banana | Промпты для Midjourney |
| heygen | AI-аватары с озвучкой |
| veo-video | Генерация видео через Veo 3.1 |
| prompt-engineer | Создание промптов |
| seo-blog-writer | SEO-статьи |
| selling-meanings | Продающие смыслы |
| swipefile | Разбор видео/контента в Notion |
| telegram | Контент для Telegram-каналов |

