# Перевод боли в желание

> © Фабрика Контента | OpenClaw Content Factory | Макс Галсон | https://t.me/galsonproai | galson.pro

## Формула

```
БОЛЬ: "У меня [проблема], из-за этого [последствия]"
         ↓
ЖЕЛАНИЕ: "Я хочу [обратное], чтобы [результат]"
```

## Процесс перевода

### Шаг 1: Записать боль как есть
```
"У меня низкий чек, нужно много клиентов, не знаю как поднять"
```

### Шаг 2: Озвучить клиенту
```
"Ты сказал, что у тебя низкий чек, нужно много клиентов, 
и ты не знаешь как его поднять. Как ты хочешь, чтобы было?"
```

### Шаг 3: Записать ответ
```
"Я хочу взять и поднять свой чек аргументированно, 
чтобы у клиента не было возражения дорого"
```

## Критерии хорошего желания

✅ Конкретное (не "хочу лучше", а "хочу X")
✅ Измеримое (цифры, сроки)
✅ Эмоциональное (чувства, состояние)
✅ На языке клиента (его слова)

❌ Абстрактное ("хочу успех")
❌ Безликое ("хочу больше")
❌ Твоими словами (ты переформулировал)

---

## Что делать с желаниями

1. **Использовать дословно** на лендингах и в контенте
2. **Формулировать обещания** продукта
3. **Создавать ядро оффера** из ключевых желаний
4. **Писать прогревы** на языке клиента
