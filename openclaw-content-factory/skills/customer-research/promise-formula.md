## Базовая формула

```
ТЫ + глагол (обещание результата) + как это будет реализовано
```

---

## Правило: Жирно и Сочно

Каждое обещание должно быть настолько убедительным,
что хочется купить даже после одной фразы.

### ❌ Плохо (сухо):
```
"Ты внедришь услугу и станешь зарабатывать больше"
```
Почему плохо: нет конкретики, нет "как", нет цифр

### ✅ Хорошо (жирно):
```
"Ты внедришь дополнительную услугу микроворонка 
и минимум в 3 раза увеличишь свой средний чек.

Благодаря этому инструменту ты сможешь приводить 
клиенту прогретых клиентов, которые сразу готовы покупать.

Предлагая такую услугу, у твоего клиента не будет 
возражения 'дорого', потому что глупо отказываться от такого."
```
Почему хорошо: есть цифра (3 раза), есть механизм (микроворонка), 
есть результат для клиента (прогретые клиенты), есть отработка возражения

## Элементы сильного обещания

### 1. Конкретный результат
```
❌ "Будешь зарабатывать больше"
✅ "Минимум в 3 раза увеличишь чек"
```

### 2. Механизм достижения
```
❌ "Научишься"
✅ "Внедришь дополнительную услугу микроворонка"
```

### 3. Почему это работает
```
❌ (ничего)
✅ "Благодаря этому инструменту ты сможешь..."
```

### 4. Отработка сомнения (опционально)
```
"У клиента не будет возражения 'дорого', 
потому что глупо отказываться от такого"
```

## Примеры по желаниям

### Желание: "Хочу продавать без мощных кейсов"

```
Ты получишь доступ к "Котлу с кейсами" —
закрытый телеграм-канал, в котором все ученики 
делятся упакованными кейсами в разных нишах.

Если твой потенциальный клиент просит кейс 
из какой-то конкретной ниши, 
ты сможешь взять его из котла и продемонстрировать.
```

### Желание: "Хочу уверенно давать результат"

```
В 90% случаев ты будешь выполнять свои обязательства 
перед клиентом минимум в 2 раза качественнее, чем делал до этого.

Ты будешь работать исключительно с одной денежной нишей, 
под которую получишь конкретные связки по настройке рекламы.

Ты просто повторяешь 1, 2, 3 по нашим инструкциям 
и больше не паришься о том, что можешь столкнуться 
с новыми для себя нюансами.
```

### Желание: "Хочу тратить меньше времени"

```
Ты будешь зарабатывать больше, 
но при этом тратить минимум в 2 раза меньше времени на работу.

Сейчас ты тратишь огромное количество времени на создание креативов,
потому что их нужно регулярно менять и тестировать.
Плюс тратишь деньги на дизайнеров и время на коммуникацию с ними.

Я выдам тебе конструктор креативов.
Каждый креатив делается без вложений 
и без привлечения дизайнеров за 5 минут.
```

### Желание: "Хочу упакованный профиль"

```
Ты создашь упакованный аккаунт таргетолога, 
который будет вызывать доверие у твоего клиента 
и на входе отрабатывать все возражения.

Мы дадим большое количество примеров упакованных профилей 
и пошаговые инструкции по тому, как внедрить эту упаковку 
и какую именно информацию выложить в блок, 
чтобы в голове у клиента остался один вопрос: куда платить.
```

### Желание: "Хочу стабильный доход"

```
Ты будешь получать желаемый доход в месяц абсолютно стабильно,
не переживая о том, будут ли у тебя деньги на жизнь в следующем месяце.

Ты внедришь инструмент микроворонки.
Он может прогревать и самостоятельно продавать продукты твоего клиента,
и для этого ему нужен будет трафик.

За счёт внедрения такого инструмента ты сможешь взаимодействовать 
со своими клиентами не в роли подрядчика, а в роли партнёра 
за процент от продаж.
```

## Процесс создания обещания

### Шаг 1: Записать желание
```
"Хочу [X]"
```

### Шаг 2: Придумать КАК ты это реализуешь
```
Гипотеза: [инструмент/метод/модуль]
```

### Шаг 3: Сформулировать результат с цифрой
```
"Ты [действие] и [измеримый результат]"
```

### Шаг 4: Объяснить механизм
```
"Благодаря [чему] ты сможешь [что]"
```

### Шаг 5: Добавить отработку сомнения (если нужно)
```
"У тебя не будет [проблемы], потому что [причина]"
```

## Чек-лист обещания

- [ ] Есть конкретный результат (цифра/состояние)
- [ ] Есть механизм (как достигается)
- [ ] Объяснено почему работает
- [ ] На языке клиента
- [ ] Нет пустых слов ("лучше", "больше", "эффективнее")
- [ ] Хочется купить от одной фразы
