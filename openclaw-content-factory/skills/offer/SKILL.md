## Главная формула

```
Результативный глагол + Идеальный результат + Скорость достижения + [Технология]
```

### Компоненты

**Результативный глагол:**
- Зарабатывать
- Научиться
- Освоить
- Похудеть
- Получить

**Идеальный результат:**
- Критериально измеримый (цифры) ИЛИ
- Понятный через состояние

**Скорость:**
- За X недель/месяцев
- Всего за X дней

**Технология (опционально):**
- Освоив технологию [название]
- При помощи [метод]
- Используя [инструмент]

## Тест "Конференция"

```
Представь: ты на конференции по твоей теме.
Произносишь свою фразу.
Все поворачиваются с реакцией "ЧТО?!"
→ Ядро работает.

Никто не поворачивается.
→ Ядро слабое, переделать.
```

## Ключевые желания vs Обычные

**Ключевые** — то, ради чего люди РЕАЛЬНО приходят:
- Зарабатывать много
- Тратить времени мало
- Получить конкретный результат

**НЕ ключевые** — второстепенные:
- "Настраивать без банов"
- "Иметь упакованный профиль"
- "Получить сертификат"

> Люди думают ТОЛЬКО О СЕБЕ. Ключевые желания = про их личную выгоду.
