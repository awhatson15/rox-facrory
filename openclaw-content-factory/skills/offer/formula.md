# Формула ядра оффера

> © Фабрика Контента | OpenClaw Content Factory | Макс Галсон | https://t.me/galsonproai | galson.pro

## Базовая формула

```
Результативный глагол + Идеальный результат + Скорость + [Технология]
```

---

## Пошаговый процесс

### Шаг 1: Собрать данные из исследований

Из карты исследования вытащить:
1. **Критериально измеримый результат** (или описание через состояние)
2. **Скорость достижения** результата
3. **3 ключевых желания** (только КЛЮЧЕВЫЕ, не второстепенные)

### Шаг 2: Сформулировать результат

**Для ниш с цифрами:**
```
Я хочу зарабатывать минимум 150 тысяч рублей
без головняка, тратить на работу максимум 2 часа в день,
выставить средний чек 30к, работать с 5-6 клиентами
```

**Для мягких ниш (через состояние):**
```
Я часто испытываю оргазмы, получаю удовольствие от секса
и занимаюсь им легко и играючи
```

### Шаг 3: Добавить скорость
```
За 4 недели / За 1-2 месяца / Всего за X дней
```

### Шаг 4: Добавить технологию (опционально)
```
Освоив технологию [название]
При помощи практик от [эксперт]
Используя готовые связки
```

### Шаг 5: Добавить тезис безопасности
```
Найди первого клиента уже во время обучения
Окупи стоимость курса до его окончания
Под присмотром профессионалов
```

## Примеры готовых ядер

### Ниша: Таргет

```
Зарабатывать 150 тысяч рублей ежемесячно, 
освоив технологию привлечения подписчиков и заявок 
для блогеров и инфобиза.

Трать на работу максимум 2 часа в день, 
получая за свою услугу от 30 тысяч рублей, 
работая с 5-6 клиентами.

Найди первого клиента уже во время обучения 
и окупи стоимость курса до его окончания.
```

### Ниша: Сексология

```
Научись испытывать яркие и регулярные оргазмы 
всего за 4 недели при помощи практик и упражнений 
от сексолога с 15-летним стажем.

Ощути себя яркой, сексуальной и раскрепощённой женщиной,
занимайся сексом легко, играючи 
и получай удовольствие наравне с мужчиной,
чувствуя себя расслабленной и отключенной от всех проблем.
```

### Ниша: Похудение

```
Похудей на 10 кг за 4 недели 
без ежедневных изнурительных тренировок, 
ежедневного подсчёта калорий и отказов от сладкого.

Под присмотром профессиональных спортсменов, 
нутрициологов и диетологов, 
благодаря которым уже более 500 женщин 
похудели в среднем на 17 кг.
```

## 3 варианта построения ядра

### Вариант 1: На желаниях
Когда есть 3 ярких ключевых желания — строим на них.

```
Ядро = Результат + Желание1 + Желание2 + Желание3
```

### Вариант 2: На сервисе/безопасности
Когда конкуренты предлагают то же самое — выделяемся сервисом.

```
Ядро = Результат + Сервис1 + Сервис2 + Сервис3
```

Пример:
```
Похудей на 10 кг:
- Без изнурительных тренировок
- Без подсчёта калорий
- Без отказов от сладкого
```

### Вариант 3: На технологии
Когда есть уникальный метод или название.

```
Ядро = Результат + Уникальная технология + Скорость
```

## Чего ИЗБЕГАТЬ

❌ "Проработай и приди к успеху" — непонятно что значит
❌ "Сделай так, чтобы стало лучше" — нет конкретики
❌ "Каждый запуск крупнее и крупнее" — эфемерно
❌ "Выведи на новый уровень" — пустые слова

> Либо критериально измеримый результат, 
> либо нормальное описание через состояние.

---

## Чек-лист ядра

- [ ] Есть конкретный результат (цифра или состояние)
- [ ] Есть срок достижения
- [ ] Проходит тест "конференция"
- [ ] Содержит только КЛЮЧЕВЫЕ желания
- [ ] Нет эфемерных формулировок
- [ ] На языке клиента (не твоими терминами)
