# Продающие Смыслы

Система извлечения продающих смыслов из экспертного контента, кейсов и исследований аудитории.

## Быстрый старт

```bash
# Извлечь смыслы из текста
"Извлеки продающие смыслы из: [текст/транскрипт]"

# Упаковать смысл в пост
"Упакуй смысл '{смысл}' в threads пост"

# Карта смыслов для продукта
"Составь карту смыслов для продукта: [название]"
```

## Что такое продающий смысл

Продающий смысл — это идея, которая:
1. Резонирует с болью/желанием аудитории
2. Позиционирует {ваш эксперт} как решение
3. Может быть упакована в разные форматы (пост, рилс, сторис, лендинг)

## Источники смыслов

| Источник | Как извлекать |
|----------|---------------|
| Кейсы клиентов | Результат + эмоция до/после |
| Возражения | Перевернуть в аргумент |
| FAQ | Частые вопросы = боли |
| Отзывы | Цитаты с конкретикой |
| Конкуренты | Что они НЕ говорят |
| Личный опыт | Истории трансформации |

## Формула смысла

```
СМЫСЛ = Боль аудитории + Инсайт эксперта + Доказательство
```

### Пример
- **Боль:** "Не могу продавать, чувствую себя впаривателем"
- **Инсайт:** "Продажа = помощь, если продукт реально решает проблему"
- **Доказательство:** "Клиент пришёл со словами 'наконец кто-то объяснил нормально'"
- **Смысл:** Продавать — значит помогать найти решение, которое человек уже ищет

## Карта смыслов

Для каждого продукта составляй карту:

1. **Смыслы входа** — почему начать сейчас
2. **Смыслы доверия** — почему именно {ваш эксперт}
3. **Смыслы результата** — что получит клиент
4. **Смыслы срочности** — почему не ждать
5. **Смыслы отстройки** — чем отличается от конкурентов

## Упаковка смыслов

Один смысл → множество форматов:

| Формат | Как упаковать |
|--------|---------------|
| Threads пост | Хук из смысла + раскрытие в 500 символов |
| Reels | Визуальная метафора смысла |
| Сторис | Опрос/вопрос вокруг смысла |
| Лендинг | Блок с заголовком из смысла |
| Email | Тема письма из смысла |

## Частые ошибки

❌ Смысл без конкретики — "мы помогаем расти" (кому? как? на сколько?)
❌ Смысл-клише — "качество по доступной цене"
❌ Смысл без доказательства — голословное утверждение
❌ Один смысл на всё — нужна карта из 15-20 смыслов
