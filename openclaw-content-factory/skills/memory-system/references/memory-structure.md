## HEARTBEAT.md — шаблон

```markdown
# Heartbeat — Автообслуживание

> Правила автоматического обслуживания памяти. Выполняй каждые 30 минут активной работы.

## Каждые 30 минут

1. **Snapshot** — запиши в `memory/YYYY-MM-DD.md`:
   - Что сделано за последние 30 минут
   - Какие решения приняты
   - Какие выводы сделаны

2. **Проверка контекста** — если контекст загружен >70%:
   - Убедись что всё важное записано в файлы
   - Ключевые решения → MEMORY.md
   - Ошибки → learning/anti-patterns.md

3. **Актуализация** — если в ходе работы появились:
   - Новые факты о владельце → MEMORY.md (секция Владелец)
   - Новые решения → MEMORY.md (секция Решения)
   - Новые грабли → MEMORY.md (секция Грабли)

## При старте нового сеанса

1. `read MEMORY.md` — восстановить контекст
2. `read memory/YYYY-MM-DD.md` — что было сегодня
3. `read learning/approved-patterns.md` — что работает
4. `read learning/anti-patterns.md` — чего избегать

## Правило

**ВСЁ записывается в файлы. Компакция сожрёт контекст — файлы останутся.**
```
