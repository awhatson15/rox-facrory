## Формат слайда "Фото + Промпт" (основной)

Один слайд = одно изображение с встроенным промптом.

### Структура слайда:
```
┌─────────────────────────┐
│                         │
│      ФОТО (верх)        │
│      ~60% высоты        │
│                         │
├─────────────────────────┤
│ [эмодзи] Название       │
│ ПРОМПТ / ОПИСАНИЕ      │
│ ┌─────────────────────┐ │
│ │ Текст промпта       │ │
│ │ в блоке с подложкой │ │
│ └─────────────────────┘ │
│     [ваш username]      │
└─────────────────────────┘
```

### Элементы:
1. **Фото** — занимает верхнюю часть слайда (~60%)
2. **Эмодзи + Название** — жирный шрифт
3. **Подзаголовок** — "ПРОМПТ / ОПИСАНИЕ" (капс, мелкий, акцентный цвет)
4. **Блок промпта** — текст в рамке/подложке
5. **Ваш username** — мелко внизу по центру
